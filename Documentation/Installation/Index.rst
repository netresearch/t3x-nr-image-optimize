============
Installation
============

Requirements
============

-  PHP 8.2, 8.3, or 8.4
-  TYPO3 13.4+
-  Intervention Image library (installed automatically via Composer)

Via Composer (recommended)
==========================

..  code-block:: bash

    composer require netresearch/nr-image-optimize

Manual installation
===================

1. Download the extension from the
   `TYPO3 Extension Repository <https://extensions.typo3.org/extension/nr_image_optimize>`__.
2. Upload to :file:`typo3conf/ext/`.
3. Activate the extension in the Extension Manager.

Recommended extensions
======================

-  `imageoptimizer <https://github.com/christophlehmann/imageoptimizer>`__
   -- additional image optimization tooling
